# Woolf University. Design Patterns Course. Homework – OOP Basics

## Homework 1 - OOP Basics

Practice core Object-Oriented Programming principles in TypeScript.

### Topic

Build a small domain model that demonstrates:

* Encapsulation
* Aggregation
* Abstraction
* Inheritance
* Polymorphism

### Requirements

The solution should include several related entities, a central aggregator, and a separate service for business logic.

### Main Logic

The app should demonstrate:

* Creating domain entities
* Connecting related objects
* Managing object state
* Reusing shared behavior through abstraction
* Extending behavior through inheritance
* Showing different behavior through polymorphism

### Notes

* Use TypeScript only.
* Do not use external libraries.
* Keep each class in a separate file.
